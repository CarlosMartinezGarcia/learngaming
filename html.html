1. Introduccion

HTML (Hyper Text Markup Language) es el lenguaje estándar para la creación de páginas webs. Mediante el uso de etiquetas podemos describir la estructura de la página web y dichas etiquetas le dicen al navegador como mostrar la web.
Inicialmente, las páginas solo mostraban el contenido HTMl, pero con el desarrollo del estándar HTMl por el W3C, las páginas empezaron a incluir también información sobre el estilo de sus contenidos (CSS) y posteriormente código de aplicaciones (JavaScript)

2. Características básicas


2.1 Estructura

Las páginas HTML tienen dos partes diferenciadas: la cabecera y el cuerpo. La cabecera incluye información relacionada con la página como puede ser el título o su idioma. El cuerpo contiene todos los elementos que se quiere se muestren al cargar la página.
La cabecera irá encerrada por la etiqueta head, mientras que el cuerpo se encerrará con la etiqueta body. Además la etiqueta html mostrará el principio y fin del documento.
Adicionalmente, existirá obligatoriamente una primera linea llamada doctype que le dice al navegador la versión de html a ejecutar. 


                 <!DOCTYPE html>   
                 <html>

                 <head>
                 <title>Mi primera Web</>
                 </head>

                 <body>
                 <p>Aprender HTML puede ser muy sencillo</p>
                 </body>

                 </html>

2.2 Etiquetas

En HTML5 existen unas 114 etiquetas, algunas de ellas ya obsoletas. Las etiquetas más usadas son: a, br, button, col,data,div,font,font,footer,h1-h6,head, header,hr,html,img,input,label,li,link,nav,ol,option,script,span,strong,style,table,td,th,thead,tr y ul. Puedes consultar información sobre las etiquetas desde el apartado de herramientas externas.
A pesar de la extensa lista de etiquetas, no es necesario para la elaboración de páginas complejas. Para ello se necesitan atributos, que a su vez son usados por CSS y JavaScript en ocasiones para cambiar el estilo y hacer la página dinámica respectivamente.


2.3 Atributos

Un atributo proporciona información adicional sobre un elemento HTML. Todos los elementos HTML pueden tener atributos que irán dentro de la etiqueta inicial y serán del tipo nombre/valor. Por ejemplo la etiqueta a, se usa para incluir enlaces en la web, y si no se le pone ningún atributo, no se puede referenciar a que dirección apunta dicho enlace. Para ello se usa el atributo href y el valor asignado será la dirección. Por ejemplo: 

             <a href="https://www.w3schools.com"> Pincha para visitar la página</a>

3. Texto

La mayoría de páginas webs, están compuestas casi en su totalidad por texto. Existen muchos etiquetas para texto, con diferentes funcionalidades, y vamos a ver algunas de las más importantes.


3.1 Párrafos

Una de las más repetidas en las páginas web es la etiqueta p. La etiqueta p, nos permite definir los párrafos que conforman nuestra página web. Los párrafos siempre empiezan en una nueva linea, y el navegador rellenará con espacios en blanco (margen) antes y después del párrafo. Importante mencionar que aunque escribamos el párrafo en varías lineas o añadamos muchos espacios, la etiqueta p hará que el navegador los ignore y se escribirá todo en una misma linea. 

                 <p>Esto es un párrafo</p>

                 <p>Esto
                 seguirá siendo un párrafo
                 escrito en una sola
                 linea 
                 </p>

Si queremos añadir una nueva linea, deberemos usar la etiqueta br. Esta etiqueta es una etiqueta "vacía", lo que significa que no hace tiene etiqueta de cierre. 

                  <p> Esto <br>si es un párrafo representado en <br> varias lineas y escrito en una sola con la etiqueta p <p>
En algunos casos, nos conviene y nos es más sencillo, que el navegador muestre el texto con el formato original, ya sea porque favorece la legibilidad del texto, o porque dicho texto debe ser procesado por alguna aplicación. En estos casos se usa la etiqueta pre. 

                  <pre> Este texto     se     muestra
                  con los espacios y las nuevas     lineas
                  tal 
                  y      como    lo escribimos
                  </pre>

Para más información sobre etiquetas para dar formato al texto como strong,ins y del , consulta la documentación del apartado herramientas externas.

3.2 Secciones

Las secciones se suelen usar para los títulos. En HTML vienen definidas las etiquetas h1,h2,h3,h4,h5,h6 para la creación de secciones. Están ordenadas de manera que h1 es la sección más importante y h6 la que menos.
Para poder ver la diferencia, puedes fijarte en que los apartados que figuran en el índice de esta web estan escritos con la etiqueta h2, mientras que los subapartados usan la etiqueta h5

4. Imagenes

Otro elemento fundamental en una página web son las imágenes ya que nos pueden mejorar notablemente el diseño. Para incluir imágenes se usa la etiqueta vacía img y requiere obligatoriamente el atributo src que es la URL de la imagen a cargar. Además se usa muy habitualmente el atributo alt, que especifica el texto alternativo a mostrar si falla la carga de la imagen. 


                                           <img src="images/logoHTML.png" alt="Logo de HTML">

5. Tablas

HTML incluye soporte de tablas desde sus primeras versiones. Estas nos permiten almacenar información en columnas y filas. Para ello se usan las etiquetas table, td,tr y th.

6. Listas

Las listas nos servirán para agrupar elementos que estén relacionados entre ellos. Los dos tipos de listas principales son las listas ordenadas y las listas no ordenadas.

6.1 Listas no ordenadas

Las listas no ordenadas es el tipo de lista mas simple. Como su nombre indica, una lista no ordenada es un conjunto de elementos relacionados entre si pero que no tienen ningun orden específico. Las etiquetas usadas para las listas no ordenadas son li y ul (list item y unordered list respectivamente) 

   Lista de colores:
   <ul>
   <li>Rojo</li>
   <li>Azul</li>
   <li>Verde</li>
   <li>Amarillo</li>
   <li>Morado</li>
   </ul>
  • Rojo
  • Azul
  • Verde
  • Amarilo
  • Morado
6.2 Listas ordenadas

Las listas ordenadas son casi idénticas pero los elementos siguen un orden concreto. Se pueden usar igualmente para listas que no tengan porque seguir un orden, como vamos a mostrar a continuación con el ejemplo de los colores para ver las diferencias más claramente.
Cambiando las etiquetas ul por ol, conseguiremos nuestra lista ordenada 

   Lista de colores:
   <ol>
   <li>Rojo</li>
   <li>Azul</li>
   <li>Verde</li>
   <li>Amarillo</li>
   <li>Morado</li>
   </ol>
  1. Rojo
  2. Azul
  3. Verde
  4. Amarilo
  5. Morado

7. Enlaces

Los enlaces fueron una de las claves del éxito de HTML ya que hizo que las páginas fueran mas interactivas. Los enlaces generalmente referencian a otras webs, pero también pueden referenciar recursos como archivos,imagenes,etc.
Los enlaces se definen con la etiqueta a. Al igual que en las imágenes se usa el atributo href para la URL. 

<a href="index.html">Pincha para volver al inicio de nuestra página</a>
