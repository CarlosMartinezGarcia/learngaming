Actualmente está en desarrollo la versión 2.0 de Dungeon Code, la cual incluirá las siguientes novedades :

  • Nuevo mapa: Mazmorra subterránea
  • 10 nuevos niveles con dificultad progresiva
  • Nuevas acciones disponibles, asi como posibilidad de resolver niveles usando el lenguaje C
  • Nuevo enemigo, Norman, que pondrá a prueba a nuestros protagonistas y el conocimiento del jugador

Además, se añadirá nuevo contenido a la pagina web :

  • Nuevos manuales para los lenguajes de JavaScript, C# y SQL
  • Profundización de los lenguajes de Java y C con más ejemplos de código
  • Cuestionarios para poner a prueba los conocimientos de los diferentes lenguajes
